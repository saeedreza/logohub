üìñ API Documentation

Base URL

https://logohub.dev/api/v1

Endpoints

Endpoint Method Description
/v1/logos GET List all available logos with pagination
/v1/logos/{id} GET Get logo metadata and format options
/v1/logos/{id}?file={name}.{format} GET Get logo file with optional customization
/health GET API health check

Parameters

  • file (required): Filename with format (e.g., standard.svg)
  • size (optional): Size in pixels for PNG/WebP (1-2048)
  • color (optional): Hex color for SVG customization (without #)
  • page (optional): Page number for logo listing
  • limit (optional): Items per page (max: 100)

Supported Formats

  • SVG: Scalable vector graphics with color customization
  • PNG: Raster format with transparency
  • WebP: Modern format with superior compression

Standard Sizes

16, 32, 64, 128, 256, 512 pixels
